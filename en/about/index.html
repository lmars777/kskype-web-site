The KSkype audience has big dreams

On July 9, 2024, in the morning we were able to fix the ReSkype 2.5 login, since then we have been developing our product under the KSkype brand. Together we will be able to continue developing, introducing new features to our already KSkype. And now we can guarantee you to keep updating the client and give you the best user experience. We are constantly working to make the use of our client more convenient and practical. And this is just the beginning.

What is KSkype?

KSkype is based on Skype for Linux version 5. It is quite similar to the familiar Skype version 7, but it is not. Unlike Skype version 7, Skype for Linux can be edited, which was done for the first time by vl0d and Pavlosik at the beginning.

In the fall of 2023, Microsoft disabled authorization in this client, but Inception Time was able to first fix the login. Later, a second Skype client appeared - based on Skype for Linux - Skype Classic.

We are constantly developing KSkype. Our goal is to get closer to the design of Skype 7.41. We regularly release updates and fixes. And now we are working on a new generation of KSkype, which will not leave anyone indifferent.
