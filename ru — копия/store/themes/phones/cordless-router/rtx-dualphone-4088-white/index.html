What is Skype Credit?

Skype Credit is the way to pay for those Skype features that are not free. You can use it to make great value calls to phones and mobiles across the world, buy an Online Number so your friends without Skype can call you at local rates, send low cost SMS messages, and forward calls. You can also use Skype Credit to buy a calling subscription, and to connect to over 200,000 WiFi hotspots worldwide using Skype WiFi.

Features:

  • Skype calls without a computer.
  • Two Skype accounts supported on multiple handsets.
  • TFT color display.
  • HD audio.
  • Full duplex speakerphone.
  • Headset jack (3.5mm).
  • New message light indicator on handset (voicemail, missed call).
  • DECT air-interface using authentication and encryption.
  • Easy to install, easy to use.
  • Skype-to-Skype calling.
  • Skype Online Number support.
  • Call forwarding and conference calling with Skype, plus call waiting.
  • Caller ID with Skype Name and picture of calling party.
  • Manual and automatic sign in to Skype account.
  • Update notification when new Skype and DUALphone software is available.
  • Can be used with up to four additional handsets.
